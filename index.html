16 modules

Ce que fait le kit

Chaque module est indépendant. Lancez tout le cycle d'un coup ou choisissez uniquement ce dont votre PC a besoin.

Diagnostic & Sauvegarde

00

Diagnostic

Inventaire complet : CPU, RAM, disques SMART, antivirus, BitLocker, pilotes en erreur, temps de boot.

01

Sauvegarde

Point de restauration système + copie des données utilisateur, OneDrive-aware.

10

Rapport

Rapport HTML avant/après chiffré : espace récupéré, apps retirées, état de santé machine.

Sécurité

02

Antivirus

Supprime Avast/AVG, active Windows Defender, lance un scan rapide après mise à jour des signatures.

08

Comptes

Compte admin séparé + compte courant en standard. Garde-fou : toujours au moins 2 admins.

13

BrowserPUP

Retire les détournements de navigateur posés par policy de registre (moteur forcé, onglets forcés).

Nettoyage & Débloat

03

Débloat

Supprime bloatware Store et OEM (Asus, Dell, HP...). Jeux Game Pass toujours protégés.

07

Nettoyage

Vide temp/cache, DISM, SFC, défrag/TRIM selon le type de disque. +8 Go récupérés en moyenne.

11

DeepClean

Raccourcis morts et dossiers résiduels post-débloatage. Ne touche jamais au registre.

Optimisation & Confort

04

Privacy

Réduit la télémétrie Windows, désactive Copilot, Recall, Widgets, Search Highlights.

05 / 06

Mises à jour & Logiciels

Windows Update + winget. Installe Firefox, 7-Zip, VLC, LibreOffice (liste configurable).

09 / 12

Confort & Startup

OneDrive désactivé, extensions visibles, désactivation réversible des autostarts indésirables.

14

Undo

Annule en LIFO les actions réversibles du dernier run. Supporte -WhatIf pour prévisualiser.

15

Réseau

Réinitialisation de la pile réseau (Winsock, TCP/IP, DNS) pour une connexion capricieuse. Non réversible, décoché par défaut.

Questions fréquentes

Avant de lancer un script admin...

Les vraies questions de sécurité méritent des vraies réponses.

Le kit commence toujours par créer un point de restauration système (module 01). Toutes les actions destructives demandent une confirmation interactive. Les fichiers personnels (photos, documents) ne sont jamais supprimés ni déplacés par aucun module.

Le mode -WhatIf vous permet de lire exactement ce qui serait fait sans rien modifier. Lancez-le d'abord pour inspecter en toute sécurité.

PowerShell 5.1 est inclus dans Windows depuis 2017 et n'exige aucune installation. C'est le seul langage natif capable d'administrer l'ensemble des couches Windows (WMI, registre, AppX, tâches planifiées) sans dépendance externe.

Surtout : le code source est lisible. Chaque ligne est visible sur GitHub. Un .exe compilé ne l'est pas.

C'est normal pour un fichier téléchargé depuis internet. Windows marque tout fichier non signé avec un "Mark of the Web" et SmartScreen l'affiche en avertissement.

Méthode 1 (recommandée) : Clic droit sur le fichier .zip téléchargé - Propriétés - cocher "Débloquer" en bas - OK. Dézipper ensuite.

Méthode 2 : Si la boîte "Windows a protégé votre PC" s'affiche, cliquer "Informations complémentaires" puis "Exécuter quand même".

Le code est consultable sur GitHub avant de débloquer quoi que ce soit.

Non. Le kit ne fait aucune connexion sortante hormis celles explicitement demandées par l'utilisateur : Windows Update (module 05) et winget pour installer des logiciels (module 06).

Aucune télémétrie, aucun envoi de logs, aucun compte requis. Tout reste sur la machine ou sur la clé USB.

Partiellement, selon les modules. Les modules 12 (Startup) et 13 (BrowserPUP) sont entièrement réversibles via le module 14 (Undo) : il lit le manifeste d'annulation écrit pendant le run et défait chaque action dans l'ordre inverse.

Le point de restauration système créé par le module 01 permet de revenir à l'état initial complet si nécessaire.

Les suppressions de bloatware (module 03) ne sont pas réversibles sans le point de restauration - raison de plus pour ne pas le sauter.
